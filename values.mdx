### GetModifierCount
Gets the current modifier count.

```lua
print(CustomModifiers:GetModifierCount())
```

### GetCategoryCount
Gets the current category count.

```lua
print(CustomModifiers:GetCategoryCount())
```

### GetFloorCount
Gets the current floor count.

```lua
print(CustomModifiers:GetFloorCount())
```
