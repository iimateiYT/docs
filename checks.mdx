### ModifierExists
Checks if a modifier exists and returns true if it exists.

<ParamField path="Modifier" type="String" required>
  Checks if a modifier exists and returns true if it exists.
</ParamField>

```lua
if CustomModifiers:ModifierExists("Example") then
    print("Exists!")
else
    print("Doesn't Exist!")
end
```
